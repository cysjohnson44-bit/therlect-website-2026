# Therlect 網站全面調整方案

## 參考 DigitalOcean 設計的調整計劃

### Phase 1: 導航和頂部菜單優化

**目標**：簡化導航結構，使用多層級下拉菜單

**調整內容**：
- 導航菜單改為：首頁 | 核心技術 | 解決方案 | 關於我們 | 聯絡我們
- 添加下拉菜單分類：
  - 核心技術 → 被動散熱、主動冷卻、液冷系統、遠紅外線
  - 解決方案 → 消費性電子、工業通訊、健康與生活、汽車與航空
- 右側操作：客服詢問、GET IN TOUCH

### Phase 2: 英雄區重構

**目標**：參考 DigitalOcean 的英雄區設計

**調整內容**：
- 標籤：「THERMAL MANAGEMENT EXPERTS」（保持）
- 大標題：「釋放熱能的無限潛力」（保持）
- 副標題：公司簡介（保持）
- 雙 CTA 按鈕：「探索解決方案」和「聯絡我們」（保持）
- 右側視覺元素：保持熱分析圖片和數據卡片
- 添加統計數據區塊：展示 3 個關鍵指標（如 7+ 年經驗、100+ 客戶、50+ 項目）

### Phase 3: 解決方案分層展示

**目標**：使用類似 DigitalOcean 的五層架構展示

**調整內容**：
- 從「我們的解決方案」改為分層展示：
  1. 被動散熱方案
  2. 主動冷卻系統
  3. 液冷技術
  4. 遠紅外線應用
  5. 整體熱管理解決方案

- 每層包含：
  - 標題和簡短說明
  - 關鍵特性列表
  - 視覺化圖表或圖片
  - 「了解更多」按鈕

### Phase 4: 卡片設計統一

**目標**：統一使用卡片設計展示內容

**調整內容**：
- 所有內容區塊使用統一的卡片設計
- 卡片包含：邊框、內邊距、陰影、懸停效果
- 使用虛線邊框突出重點區塊
- 統一的間距和對齊

### Phase 5: 資源和文章展示

**目標**：添加類似 DigitalOcean 的資源區塊

**調整內容**：
- 創建「最新資源」或「技術文章」區塊
- 使用網格佈局展示文章卡片
- 每個卡片包含：標題、發布日期、閱讀時間、簡短描述
- 添加「查看全部」按鈕

### Phase 6: 響應式設計優化

**目標**：確保在所有屏幕尺寸上都有良好的顯示效果

**調整內容**：
- 導航在移動端收縮為漢堡菜單
- 多列佈局在小屏幕上變為單列
- 字體大小根據屏幕調整
- 按鈕和交互元素在移動端易於點擊

## 保持不變的內容

- 所有文字內容
- 所有圖片和視覺元素
- 顏色方案（深色背景、青色強調）
- 品牌標識和 Logo
- 公司信息和聯絡方式

## 實施順序

1. 分析現有結構
2. 優化導航菜單
3. 重構英雄區
4. 調整解決方案展示
5. 統一卡片設計
6. 添加資源區塊
7. 測試和驗證
8. 保存檢查點
